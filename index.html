what we used

In this project, we used these tools to build some interactive data visualizations which can efficiently provide necessary information for voters to make their decisions.

Python

Python has been used for data scraping and cleaning. It provides us with a great way to gain more data, and process data more efficiently.

Tableau

Using Tableau Public and Tableau Desktop for analysis. We aim to provide a set of intuitive and interactive visualization to help users create more penetration for data.

Web Design

Last but not least, we produced a website to adequately represent our visualizations.
